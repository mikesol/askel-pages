Skill Taxonomy

Routing model: The top-level playbook skill is a dispatcher. It reads context (deal stage, task type) and surfaces only the relevant sub-skill. Cron jobs load playbook-thesis directly; interactive sessions route through playbook. Progressive disclosure — don't load the whole playbook, load what's relevant.
